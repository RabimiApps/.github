English:

# 🌟 RabimiApps

**RabimiApps** is a personal app development brand by **Rabimi**,  
mainly focused on **Android apps** and **Minecraft-related tools**.  
The goal is to make it possible to manage and play Minecraft Java  
freely even on a smartphone, and we continue development every day.

---

## 🧩 Main Projects

### 🎨 [JavaSkinChanger (JSC)](https://github.com/RabimiApps/JavaSkinChanger)
An **Android app that allows you to change Minecraft Java Edition skins**.  
Designed for PojavLauncher users and players who want to play Java Edition without a PC.

- 📱 Language: Kotlin  
- ⚙️ Build: Gradle 8.7  
- 🔄 Auto Build & Release: GitHub Actions  
- 🧾 License: Apache-2.0  

---

### 🌍 [mcserverstatus](https://github.com/RabimiApps/mcserverstatus)
**Android tool to check the status of Minecraft servers.**  
Lightweight and fully functional on mobile devices, it forms the core of RabimiApps' custom server monitoring system.

---

## 🛠 Technology Stack

- **Languages:** Kotlin
- **Development Environment:** Github Android, VSCode, Github for Web, Termux  
- **CI/CD:** GitHub Actions (Auto-build, signing, release)  
- **Target OS:** Android (primary), some tools also support Windows  

---

## 💡 Vision

> “To enable complete freedom to operate, develop, and manage Minecraft, even on a smartphone.”

RabimiApps focuses on providing **PC-level freedom on Android**,  
especially for Minecraft Java Edition and server-related tools.
